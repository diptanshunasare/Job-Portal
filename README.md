# Job-Portal
Job Nest
Job Nest is a comprehensive job portal designed to connect job seekers with employers, providing a seamless experience for job applications and recruitment processes.

Features
User Authentication and Authorization: Secure login and registration for job seekers and employers.
Job Listings and Applications: Employers can post job vacancies, and job seekers can browse and apply for jobs.
Profile Management: Users can create and manage their profiles, including resumes and cover letters.
Job Recommendations: Personalized job recommendations based on user profiles and preferences.
Messaging System: Direct communication between job seekers and employers.
Admin Dashboard: Admin functionalities to manage users and job listings.
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Java, JSP, Servlets
Database: MySQL
Libraries: JSTL for JSP
Tools: Apache Tomcat, Maven
